개인정보처리방침

시행일: 2026년 2월 11일 | 최종 수정일: 2026년 2월 11일

J-news는 사용자의 개인정보를 수집하지 않으며, 모든 데이터는 사용자의 기기에만 저장됩니다.

1. 개요

J-news(이하 "본 앱")는 AI 기반 뉴스 브리핑 서비스를 제공하는 모바일 애플리케이션입니다. 본 개인정보처리방침은 본 앱이 사용자 정보를 어떻게 처리하는지 설명합니다.

2. 수집하는 정보

본 앱은 개인정보를 수집하지 않습니다. 회원가입, 로그인, 계정 생성이 없으며, 사용자를 식별할 수 있는 어떠한 정보도 서버로 전송하지 않습니다.

기기에 로컬 저장되는 데이터

아래 데이터는 사용자의 기기(SharedPreferences)에만 저장되며, 외부 서버로 전송되지 않습니다.

데이터 용도 저장 위치
뉴스 캐시 오프라인에서 마지막 뉴스 열람 기기 로컬
북마크 사용자가 저장한 뉴스 보관 기기 로컬
구독 상태 Pro 기능 잠금해제 여부 기기 로컬
앱 설정 알림 스케줄 등 사용자 설정 기기 로컬

3. 앱 권한

본 앱은 다음 권한을 사용합니다.

권한 용도 필수 여부
인터넷 (INTERNET) 뉴스 데이터 서버에서 수신 필수
알림 (POST_NOTIFICATIONS) 뉴스 브리핑 알림 발송 선택

4. 데이터 전송 및 공유

  • 본 앱은 사용자 개인정보를 제3자와 공유하지 않습니다.
  • 뉴스 데이터 요청 시 서버와 HTTPS 암호화 통신을 사용합니다.
  • 서버는 사용자 식별 정보(IP, 기기 정보 등)를 저장하지 않습니다.

5. 광고 및 분석

본 앱은 현재 광고 SDK 및 분석 도구를 포함하고 있지 않습니다. 향후 도입 시 본 방침을 업데이트하겠습니다.

6. 아동 보호

본 앱은 만 13세 미만의 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

7. 데이터 삭제

본 앱의 모든 데이터는 기기 로컬에만 저장되므로, 앱을 삭제하면 모든 데이터가 자동으로 삭제됩니다. 별도의 데이터 삭제 요청은 필요하지 않습니다.

앱 삭제 전 개별 데이터를 삭제하려면:

  • 북마크: 북마크 화면에서 스와이프하여 개별 삭제
  • 전체 데이터: 기기 설정 > 앱 > J-news > 저장공간 > 데이터 삭제

8. 보안

본 앱은 다음과 같은 보안 조치를 적용하고 있습니다.

  • 모든 네트워크 통신에 HTTPS(TLS) 암호화 적용
  • 서버 API에 인증 및 접근 제어 적용
  • 사용자 데이터의 기기 외부 전송 없음

9. 개인정보처리방침 변경

본 방침은 법률 또는 서비스 변경에 따라 수정될 수 있습니다. 중요한 변경 시 앱 내 공지 또는 본 페이지를 통해 안내합니다. 변경된 방침은 본 페이지에 게시된 시점부터 효력이 발생합니다.
